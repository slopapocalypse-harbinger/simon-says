# Simon Says: Mindbreaker Odyssey

A high-production single-page strategy/idle game made for long sessions.

## Features
- Multi-zone progression with scaling danger/reward
- Energy-based tactical actions with risk/reward combat outcomes
- Upgrade economy with compounding passive gains
- Artifact forging with permanent bonuses
- Dynamic quests and timed challenges
- Achievements and progression milestones
- Save/load with localStorage
- Rich event feed and responsive UI

## Run locally
Because this is a static web app, you can run it with any static server.

Example:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
